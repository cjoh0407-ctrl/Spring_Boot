### 문자열 길이
***

*String*은 *int*나 *boolean*과 같은 기본 타입이 아니라, *char* 값들의 배열을 포함하는 객체입니다. 이는 String 클래스 내부 어딘가에서 원래 Java 개발자들이 String 객체에 대한 인스턴스 메서드들도 선언했음을 의미합니다.

다시 말해, 문자열에 특별한 메서드를 적용할 수 있습니다. 다음은 두 가지 예시입니다:

- `.repeat()`: `"Java".repeat(3)`는 *"JavaJavaJava"*를 반환합니다.
- `.length()`: `"Java".length()`는 *4*를 반환합니다.

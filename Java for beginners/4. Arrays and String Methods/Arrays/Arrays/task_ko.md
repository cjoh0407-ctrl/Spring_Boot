### 배열  
***

동일한 범주의 여러 변수가 있을 때 각각을 개별적으로 저장하고 다루는 것은 비효율적일 수 있습니다.  
대신, 같은 유형의 값을 특정 순서로 포함하는 목록을 만드는 것이 편리합니다. 이 목록은 Java에서 *배열*이라고 합니다.  

배열을 선언하려면 저장할 변수의 유형, 대괄호 `[]` 쌍, 그리고 배열의 이름을 지정합니다:

`<variable_type>[] <array_name>`

선언된 배열을 초기화하는 방법은 두 가지가 있습니다.  

첫 번째 방법은 배열을 인스턴스화하는 것입니다:

`new <variable_type>[<amount_of_elements_in_the_array>]`

그리고 배열 내부의 위치를 인덱스로 지정하여 수동으로 각 요소를 추가합니다. 문자열과 마찬가지로, 목록의 첫 번째 요소는 인덱스 0으로 표시됩니다.  

`<array_name>[<index>] = <value>`

배열을 초기화하는 다른 방법은 중괄호 `{}` 안에 쉼표로 구분된 요소 목록을 설정하는 것입니다.

`<array_name> = {<list_of_values>}`

배열의 요소에 접근하려면, 해당 인덱스를 사용하여 호출합니다.

### 정적 속성
***

일정 유형의 모든 객체가 공유하는 속성이 있습니다.

예를 들어, 모든 비행기는 크기나 색상이 다르지만, 모두 두 개의 날개가 있습니다.

이와 같은 경우에는 이러한 공유 값을 저장하기 위해 다른 유형의 속성을 사용해야 합니다.  
이 속성을 *정적* 속성이라고 합니다.

속성을 *정적*으로 만들기 위해서는 접근 제어자(*public*) 뒤에 해당 키워드를 배치하면 됩니다.

물론, 정적 속성을 변경하는 메서드를 사용하고 싶을 수도 있습니다. 이 경우, 해당 메서드들 역시 *정적* 메서드라고 불리며, 키워드 *static* 역시 접근 제어자(public) 뒤에 배치되어야 합니다.

항상 그렇듯이, 이 주제를 이해하는 데 코드는 도움이 될 수 있습니다.

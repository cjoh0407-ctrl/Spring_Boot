### void 메서드
***

객체를 사용하는 또 다른 유용한 작업은 특정 명령을 실행하도록 만드는 것입니다. 이를 인스턴스 메서드라고 합니다.

메서드는 본질적으로 메서드의 이름이 호출(또는 *호출됨*)될 때만 실행되는 코드 조각입니다.

코드 블록을 실행하는 것 외에도, 메서드는 값을 반환(또는 제공)하는 속성을 가지고 있습니다. 그리고 Java의 다른 변수와 마찬가지로, 반환받는 값의 타입을 명확히 지정해야 합니다.

구문은 다음과 같습니다:

`public <반환_타입> <메서드_이름> (인자)
{ 호출될 때 실행할 코드 }`

여기서 인자는 코드의 논리를 수행하는 데 필요할 수 있는 특정 값들입니다.  
인자는 필수는 아니지만, 인자를 요구하지 않더라도 괄호()를 반드시 작성해야 합니다. 괄호는 모든 메서드 구문의 일부이기 때문입니다.

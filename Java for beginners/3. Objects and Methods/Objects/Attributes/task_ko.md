### 속성
***

객체는 값을 저장하고 명령을 실행할 수 있습니다.  
전자는 속성이고 후자는 메서드입니다.

객체의 개념은 객체마다 공통된 특성의 다른 값을 가진다는 것입니다.

예를 들어, 모든 자동차(Car)는 색상과 브랜드를 가지고 있지만, 각 자동차의 값은 서로 다릅니다.

인스턴스 속성을 선언하려면 **main** 메서드 **외부**에 변수를 생성하십시오.  
Java는 구문을 인식하며, 생성하는 모든 객체는 이 변수의 복사본을 가지게 되며, 각 객체에 다른 값을 저장할 수 있습니다.

속성의 값을 접근하려면 객체의 이름을 참조한 후 점(.)을 입력하고 속성의 이름을 입력하십시오.  
이 값을 일반 변수처럼 수정할 수 있지만, 수정 시 참조하는 객체에 대해 해당 속성의 값을 변경한다는 점을 기억하십시오.

첨부된 코드는 방금 언급한 개념을 설명하기 위해 작성된 것입니다.

#### 참고: *private*이라는 단어는 해당 속성이 이 클래스에서만 접근할 수 있음을 의미합니다.

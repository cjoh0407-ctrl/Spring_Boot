### 객체
***

Java는 객체 지향 프로그래밍 (OOP) 언어입니다.

이는 Java에서 모든 클래스가 동일한 이름의 객체에 대한 청사진 역할을 한다는 것을 의미합니다.

새 객체를 선언하는 구문은 다음과 같습니다:

`<object_type> <object_name>;`

객체를 초기화하려면 *new*라는 단어를 사용하여 생성해야 합니다:

`<object_name> = new <class_name>();`

다음과 같이 선언과 초기화를 동시에 할 수도 있습니다:

`<object_type> <object_name> = new <class_name>();`

이는 Unit 1에서 봤던 기본 데이터 타입 (`int`, `double`, `boolean`)의 선언과 초기화와 유사합니다. 구조와 구문이 동일합니다.

제공된 예제가 큰 도움이 되길 바랍니다.

#### 참고: Java에서 객체는 *인스턴스*라고도 하며, 객체를 생성하는 과정을 *인스턴스화(instantiation)*라고 합니다.

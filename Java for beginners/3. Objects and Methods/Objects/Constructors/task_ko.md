### 생성자  
***  

객체를 생성하고 변수에 값을 할당하는 과정이 얼마나 반복적인지 주목하십시오. 이를 해결하는 구조가 있는데, 그것이 바로 생성자입니다.  

생성자는 속성에 할당하고자 하는 값들이 이미 정해져 있을 때, Java가 객체를 생성하는 방법을 알려줍니다.  

구문은 다음과 같습니다.  

`public <클래스_이름> ( <매개변수> )  
{ 값 할당 }`  

절대로 main 메서드 안에 생성자를 넣지 마십시오.  

왼쪽의 코드를 학습하세요.  

#### 참고: *public* 키워드는 클래스이든 속성이든 상관없이 다른 클래스에서 접근할 수 있도록 허용합니다.

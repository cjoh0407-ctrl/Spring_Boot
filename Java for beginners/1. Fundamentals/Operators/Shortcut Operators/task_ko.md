### 단축 연산자
***

우리가 다룰 마지막 숫자 연산자는 기존에 사용했던 산술 연산자의 짧은 표기법이라고 할 수 있습니다.

만약 x에 3을 더하고 싶다면, 다음과 같이 입력하는 대신

`x = x + 3;`

다음과 같이 작성할 수 있습니다:

`x += 3;`

코딩에서 매우 흔히 발생하는 경우는 정수에 1을 더하는 것입니다.  
만약 y에 1을 더하고 싶다면, 다음과 같이 입력하는 대신

`y = y + 1;`

다음과 같이 코딩할 수 있습니다:

`y++;`

그리고 동일한 표기법은 변수에서 1을 빼는 경우에도 적용됩니다.

아래 표는 이를 요약한 것입니다:

| 긴 표기법       | 단축 표기법      |
|-----------------|------------------|
| `a = a + 1;`    | `a++; `          |
| `b = b - 1;`    | `b--; `          |
| `c = c + 2;`    | `c += 2; `       |
| `d = d - 3;`    | `d -= 3;  `      |
| `e = e * 4; `   | `e *= 4;  `      |
| `f = f / 5;  `  | `f /= 5; `       |
| `g = g % 6; `   | `g %= 6;    `    |

마지막 연산자는 *모듈로* 연산자라고 부르며, 이는 나눗셈을 수행한 후의 나머지를 반환합니다.

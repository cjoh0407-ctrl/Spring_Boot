### 비교 연산자  
***

두 숫자를 비교하면 *true* 또는 *false*의 결과를 얻게 됩니다.  
비교 연산자는 수학에서 사용되는 것과 동일하게 작동합니다:

| 연산자                          | 자바에서의 표기법 |
|--------------------------------|---------------|
| 같음                           | `==`          |
| 다름 / 같지 않음                | `!=`          |
| 작음                           | `< `           |
| 작거나 같음                     | `<= `          |
| 큼                             | `> `           |
| 크거나 같음                    | `>=`           |

첫 번째 두 연산자는 숫자 값뿐만 아니라 boolean 값에도 적용할 수 있다는 점에 주의하세요.

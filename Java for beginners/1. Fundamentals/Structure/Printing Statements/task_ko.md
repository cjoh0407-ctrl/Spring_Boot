### 콘솔에서 출력하기
***
메인 메서드 내부에는 두 가지 유형의 출력문을 넣을 수 있습니다.

| 명령어                 | 효과                                                |
|----------------------|------------------------------------------------------|
| `System.out.print()`   | 콘솔에 메시지를 출력합니다.                          |
| `System.out.println()` | 콘솔에 메시지를 출력한 후 줄을 바꿉니다.               |
|                      |                                                      |
이와 같은 명령어 뒤에는 항상 라인의 끝에 세미콜론 \;을 추가해야 합니다.

*println* 명령어를 빠르게 입력하려면 단축키 *sout*을 사용할 수 있습니다.

라인을 복제하려면 단축키 &shortcut:EditorDuplicate;를 사용할 수 있습니다.

Java 코드를 실행하는 방법은 여러 가지가 있습니다:
- 클래스 또는 *main* 메서드의 왼쪽에 있는 초록색 삼각형을 클릭할 수 있습니다.
- .java 파일을 오른쪽 클릭하고 **Run**을 클릭할 수 있습니다.
- 툴바에서 *Run* 탭을 클릭하고 **Run**을 선택할 수 있습니다.
- 단축키 &shortcut:Run;를 사용할 수 있습니다.

---
##### 참고: 컴파일러는 명시적으로 지시하지 않으면 메시지들 사이에 공백을 삽입하지 않습니다.

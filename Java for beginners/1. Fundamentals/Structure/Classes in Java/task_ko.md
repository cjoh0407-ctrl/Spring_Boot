자바 소개  
=========================================================  
___  
***  
#### 이 과정은 프로그래밍 언어에 대한 개요를 제공하는 것을 목표로 합니다. 주요 구조를 예제 및 연습 문제와 함께 설명하며, 이를 통해 이해도를 테스트할 수 있습니다.  

##### 대만의 신주국제학교 9학년 학생들을 위해 원래 제작되었습니다.  

모든 자바 파일은 .java로 끝나며, 동일한 이름의 클래스를 포함해야 합니다.  

자바 파일이 무언가를 수행하도록 하려면 해당 클래스 안에 *main* 메서드를 추가해야 합니다.  

중괄호 \{ \} 는 클래스나 *main* 메서드와 같은 구조의 범위를 나타냅니다.  

실행하고자 하는 코드를 main 메서드 안에 작성하세요.  

이 파일은 간단한 자바 프로그램의 적절한 구문을 보여줍니다.  
---  
##### 참고: IDE(인텔리제이)는 .java 파일을 JVM이 이해하고 실행할 수 있는 .class 파일로 컴파일하는 작업을 처리합니다.

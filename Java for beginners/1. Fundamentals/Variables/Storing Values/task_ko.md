### 데이터 저장
***
Java 프로그램 안에 정보를 저장할 수 있습니다.  
이를 위해 먼저 Java에 어떤 유형의 데이터를 포함하는지 알려주어야 합니다.  
가장 많이 사용되는 데이터 유형은 다음과 같습니다:  
- *int*: 정수(소수점이 없는 숫자)를 저장하기 위한 데이터 유형  
- *double*: 소수점이 있는 숫자를 저장하기 위한 데이터 유형  
- *boolean*: true/false 값을 저장하기 위한 데이터 유형  
- *String*: 문자 메시지를 저장하기 위한 데이터 유형  

변수를 선언하는 구문은 다음과 같습니다:  

`<데이터 유형> <변수 이름>;`  

변수가 선언된 후, 변수의 값을 설정하려면 다음과 같이 작성하십시오:  

`<변수 이름> = <변수 값>;`  

변수를 선언과 동시에 초기화할 수도 있습니다:  

`<데이터 유형> <변수 이름> = <변수 값>;`  

이 코드에는 이를 보여주는 예제가 포함되어 있습니다.  
***
##### 참고: 수학과 다르게 코딩에서는 순서가 중요합니다. 5번째 줄을 *int 24 = hoursInADay;*로 작성할 수 없습니다. *=* 기호 양쪽의 값은 서로 바꿀 수 없습니다.

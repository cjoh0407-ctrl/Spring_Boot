### 변수의 값 변경
***
변수의 값은 언제든지 업데이트할 수 있습니다.  
대입 연산자 *=*를 사용하여 새 값을 설정하세요.

**동일한 값을 다시 선언하지 마세요.**  
Java는 동일한 이름으로 두 개의 다른 변수를 생성하려 한다고 생각하며, 이는 허용되지 않습니다.

수학 연산 및 연산자 우선순위의 일반적인 수학 규칙을 사용하세요.  
다음 표는 우선순위가 높은 순서대로 기본 산술 연산자를 요약한 것입니다.

| 연산            | 연산자     |
|-----------------|----------|
| 그룹화          | `( )`       |
| 곱셈            | `*`         |
| 나눗셈          | `/`         |
| 덧셈            | `+`         |
| 뺄셈            | `-`         |
***
##### 참고: print 문 내에서 탈출 문자 `\\n`은 Java에게 새 줄을 도입하라는 신호를 줍니다. 이는 키보드에서 &shortcut:EditorStartNewLine;을 누르는 것과 비슷합니다.

### ArrayList 메서드
***

이 과정에서는 모든 ArrayList가 기본 데이터 타입이 아닌 객체만 저장하기 때문에 wrapper 클래스에 대해 설명합니다.

이제 ArrayList를 사용할 때, 숫자 목록을 저장하려면 `ArrayList<int>`가 **아니라** `ArrayList<Integer>`를 사용하여 선언해야 함을 기억하며 사용할 수 있습니다.

좋아요. ArrayList의 유용성은 그 유연성과 메서드에 있으며, 그 중 일부는 아래 표에 요약되어 있습니다.

| 메서드                | 설명                                                                                  |
|---------------------|--------------------------------------------------------------------------------------|
| get(index)          | 해당 인덱스의 값을 반환합니다.                                                       |
| indexOf(element)    | 요소가 처음 나타나는 인덱스를 반환하거나, 요소를 찾을 수 없는 경우 -1을 반환합니다.     |
| set(index, element) | 해당 인덱스에 해당 요소의 값을 설정합니다.                                           |
| remove(element)     | 요소의 첫 번째 발생을 제거합니다.                                                     |
| remove(index)       | 해당 인덱스의 요소를 제거합니다.                                                     |
| clear()             | ArrayList의 모든 요소를 제거합니다.                                                  |
| size()              | ArrayList에 있는 요소의 수를 반환합니다.                                             |

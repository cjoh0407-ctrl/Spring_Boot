### 2D 배열
***

배열의 배열을 만들면 어떤 일이 발생할까요? 바로 2D 배열이 만들어집니다.

'2-dimensional(2차원)'의 줄임말로 2D라고 불리며, 그 이유는 배열의 첫 번째 수준을 좌표(y)로, 각 내부 배열을 두 번째 좌표(x)에 존재하는 값으로 간주할 수 있기 때문입니다.

2D 배열을 선언하는 문법은 다음과 같습니다:

`<dataType>[][] <2DArrayName>`

그리고 예상대로 초기화하려면 다음과 같은 문법을 사용합니다:

`<2DArrayName> = new <dataType>[<rows>][<columns>]`

그 후 필요시 인덱스와 하위 인덱스를 참조하여 요소를 입력하면 됩니다.

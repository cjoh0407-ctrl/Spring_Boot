### *for* 루프
***

모든 프로그래밍 언어의 기본 구조 중 하나는 루프입니다.

본질적으로, 프로그램이 이러한 선언에 도달할 때마다 내부의 코드를 여러 번 실행하게 됩니다.

*for* 루프는 다음과 같은 구문을 따릅니다:

**for ( *\<루프 변수 초기화>*; \<루프를 반복할 조건>; \<루프 변수 업데이트> )
{
반복할 코드
}**

*for* 루프는 먼저 루프 변수를 초기화한 후 코드를 한 번 실행합니다.  
그다음 변수 값을 업데이트하고 조건을 확인합니다. 조건이 참이면 코드 블록을 다시 실행하고, 다시 변수를 업데이트한 후 조건을 확인하는 과정을 반복합니다.  
조건이 *false*와 같아질 때 루프는 종료됩니다.
